目标文件就是.obj和.o文件（在不同的系统下

ELF类下的四种文件

  • 可重定向文件（Relocatable File） 包含代码和数据 静态链接库也属于这一类
  • 可执行文件（Executable File） /bin/sh文件 可执行文件，没有扩展名
  • 共享目标文件 （share Object File) 包含代码和数据。库文件以.so结尾

    即链接器
    动态链接器 可生成镜像（docker？）

可以file，命令查看文件可使用的权限

文件格式

  • 链接视图
  • 执行视图

    文件类型

  • e_ident 用于识别其为EIF-？？位文件 并且保留未使用字节的开始地址，置为0，使得处理目录忽略这些值，如果改变，则引起payload的改变
  • e_type 标识文件类型（只是标识，不会改变
  • e_machine

程序的头部只有对于可执行文件和共享目标文件有意义。
